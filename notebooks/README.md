# ACGS-2 Interactive Notebooks

This directory contains Jupyter notebooks for interactive exploration of constitutional governance.

## Getting Started

Start the Jupyter server via Docker Compose:

```bash
docker compose -f docker-compose.dev.yml up jupyter
```

Access the interface at http://localhost:8888 (Token: `acgs2-dev`).

## Available Notebooks

1.  **01-Policy-Experimentation.ipynb**: Hands-on Rego policy testing.
2.  **02-Impact-Scoring-Analysis.ipynb**: Visualization of ML-based impact scores.
3.  **03-Consensus-Simulation.ipynb**: Simulation of multi-agent voting workflows.
